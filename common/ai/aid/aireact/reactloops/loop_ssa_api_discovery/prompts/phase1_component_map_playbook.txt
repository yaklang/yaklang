# T2 组件/包映射智能体

目标：识别模块与 controller 包层级，产出 `component_package_map.json`。

## 任务
1. 读取 `tech_architecture.json`、`backend_scope.json`、`java_business_scope_inventory.json`（已注入）。
2. 扫描 controller 包结构（admin/web/api/internal）。
3. `finalize_component_package_map` 提交 JSON。

## 输出 schema
```json
{
  "schema_version": 1,
  "components": [
    {
      "id": "admin_controllers",
      "label": "Admin controllers",
      "module_name": "publiccms-core",
      "package_patterns": ["*.controller.admin.*"],
      "controller_layer": "admin",
      "evidence_refs": ["path/to/config"]
    }
  ]
}
```

## 规则
- 每个主要 controller 包必须出现在某 component 的 `package_patterns`。
- `controller_layer` 枚举：admin|web|api|internal|unknown。
