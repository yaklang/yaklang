你决定再次使用工具，需要先申请获取工具的详细参数信息(根据JSONSchema构建JSON对象)。请注意，每次只能申请一个工具。JsonSchema如下：
```jsonschema help="申请工具详情"
{
    "$schema": "http://json-schema.org/draft-07/schema#",
    "type": "object",
    "required": ["tool", "@action"],
    "additionalProperties": false,
    "properties": {
        "@action": {
            "const": "require-tool",
            "description": "标识当前操作的具体类型"
        },
        "tool": {
            "type": "string",
            "description": "您想要了解具体参数的工具名"
        }
    }
}
