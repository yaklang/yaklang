# 输出要求Schema

```schema
{
  "$schema": "http://json-schema.org/draft-07/schema#",
  "type": "object",
  "required": ["short_summary", "long_summary"],
  "properties": {
    "@action": {
      "type": "string",
      "const": "summary"
    },
    "short_summary": {
      "type": "string",
      "maxLength": 200,
      "description": "200字内的简短版本的总结"
    },
    "long_summary": {
      "type": "string",
      "description": "比short_summary更长的总结的版本。"
    }
}
```
