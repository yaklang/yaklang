使用说明

API密钥用于访问AIBalancer服务。每个密钥可以通过以下方式使用：

  • 添加到请求头: Authorization: Bearer YOUR_API_KEY
  • 作为查询参数: ?api_key=YOUR_API_KEY

为了安全起见，建议在生产环境中使用请求头方式。

警告

刷新 TOTP 密钥将使所有使用 memfit- 模型的客户端需要重新获取密钥。请谨慎操作。

使用说明

  1. 客户端通过 /v1/memfit-totp-uuid 获取包装后的 UUID
  2. 使用该 UUID 作为密钥，通过 TOTP 算法生成 6 位验证码
  3. 访问 memfit- 模型时，在请求头中添加 X-Memfit-OTP-Auth: {base64编码的验证码}
  4. 验证码有效期为 30 秒，允许 60 秒的时间误差
  5. 如果认证失败，客户端应重新获取 TOTP 密钥

Web Search 全局配置

配置全局代理，适用于所有未单独设置代理的 Web Search API Key。

CURL 调用示例:

curl -X POST https://aibalance.yaklang.com/v1/web-search \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"query": "Yaklang", "max_results": 10}'

说明: 付费用户使用 API Key 认证并计费；免费用户通过 Trace-ID 使用（需开启免费用户开关）。 searcher_type 支持: brave, tavily, chatglm, bocha, unifuncs（留空自动选择）

开启后，没有 API Key 的用户可通过 Trace-ID 免费使用 Web Search（限频：1秒/次，成功后3秒冷却）
优先级：Key 单独设置的代理 > 全局代理。全局代理仅在 Key 未设置代理时生效。

高德地图代理配置

高德地图 REST API 透明代理。 客户端请求 /amap/v3/... 将被转发到 restapi.amap.com/v3/...，服务端自动注入 API 密钥。

CURL 示例:

curl "https://aibalance.yaklang.com/amap/v3/weather/weatherInfo?city=110000" \
  -H "X-Memfit-OTP-Auth: BASE64_TOTP_CODE"

注意: 需要 TOTP 认证。免费用户有频率限制（最低间隔 1 秒/次）。

允许仅有 TOTP 认证、无 API 密钥的用户使用高德代理（限频: 最低间隔 1 秒/次）
