警告

刷新 TOTP 密钥将使所有使用 memfit- 模型的客户端需要重新获取密钥。请谨慎操作。

使用说明

  1. 客户端通过 /v1/memfit-totp-uuid 获取包装后的 UUID
  2. 使用该 UUID 作为密钥，通过 TOTP 算法生成 6 位验证码
  3. 访问 memfit- 模型时，在请求头中添加 X-Memfit-OTP-Auth: {base64编码的验证码}
  4. 验证码有效期为 30 秒，允许 60 秒的时间误差
  5. 如果认证失败，客户端应重新获取 TOTP 密钥

Web Search 全局配置

配置全局代理，适用于所有未单独设置代理的 Web Search API Key。

CURL 调用示例:

curl -X POST https://aibalance.yaklang.com/v1/web-search \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"query": "Yaklang", "max_results": 10}'

说明: 付费用户使用 API Key 认证并计费；免费用户通过 Trace-ID 使用（需开启免费用户开关）。 searcher_type 支持: brave, tavily, chatglm, bocha, unifuncs, qwen（留空自动选择）

开启后，没有 API Key 的用户可通过 Trace-ID 免费使用 Web Search（限频：1秒/次，成功后3秒冷却）
优先级：Key 单独设置的代理 > 全局代理。全局代理仅在 Key 未设置代理时生效。

模型级限流覆盖

为特定模型单独配置 RPM 上限、免费用户调用前延迟区间、输出 TPS，优先级高于全局默认值。可参考上方"热门模型 RPM"榜单设定合理上限。
RPM 留空使用全局默认；延迟最小/最大值留空使用全局默认；TPS 留空或 0 表示不限速。
延迟"最大值"留空或填 0 时按老语义 N~2N（"最小值"=N → 实际 N~2N 随机）。
模型名称请填写对外名称（如 memfit-standard-free），而非转发后的内部名称。

全局默认 RPM

每个 API Key 在一分钟内允许的最大请求数量。默认 600。
注意: 所有免费 AI 用户（-free 模型）共用 key = "free-user"，每个模型有独立的 RPM 计数桶。模型级 RPM 覆盖可为不同模型设置不同限制，互不影响。

免费用户调用前延迟

免费用户每次请求转发到上游之前，服务端会在 [最小值, 最大值] 秒之间随机 sleep（含两端）。
例如最小值 0、最大值 5 → 0~5 秒随机延迟，让用户体验更平滑；最小值与最大值都为 0 则不延迟。
兼容提示：最大值留空或填 0 时自动按老语义 N~2N（即最小值 = N → 实际 N~2N 随机）。

免费用户输出 TPS 限速

在流式响应下对免费模型按 token 数节流，上游若返回 50 token/s，可配置成 20 token/s 强制减速。
设为 0 表示不限速。模型级可单独覆盖（见本页上方"模型级限流覆盖"）。

免费用户 Token 日限额

所有免费用户（-free 模型）共享一个日 Token 池，按上游 SSE 末帧 usage 经模型四维倍率（输入/输出/缓存创建/缓存命中）加权后扣费。
基准：0.1 RMB / 1M Token。默认 1200M（约合 120 RMB/天）。
支持模型级覆盖：可为某个模型单独设定独立桶（不消耗全局池），或将该模型整体豁免（不计费）。
超额返回 429 daily_token_limit_exceeded，含 tokens_used / tokens_limit 字段。
当前已用：-- M / -- M
模型级覆盖 填限额(M)=独立桶；勾选「不计费」表示该模型完全豁免（不进入任何桶）。
全局共享池软限额 当全局共享池累计 token 用量到达软限额阈值后，仅免费模型走共享池的请求会被进一步降速到 "软限额 TPS"。模型独立桶 / 豁免模型不受影响。两项任一为 0 即关闭该机制。
单 IP 免费用量每日限额 按真实客户端 IP（支持 nginx / Cloudflare 反代头）统计每天对「计费免费模型」的请求次数与加权 Token， 防止单个 IP 高频盗刷公共免费接口、侵占其他用户额度。豁免（不计费）模型不参与统计。
请求次数或 Token 任一超限即拦截，返回 429 free_ip_limit_exceeded，提示"当前环境免费用量已用尽，请自行配置 AI 后端使用"。
每日北京时间 06:00 随免费 Token 限额一起清零。两个上限填 0 表示该维度不限制。

Memfit 客户端版本控流

仅对 memfit-* 模型生效。客户端会在每个请求带上 X-Yak-Version 与 X-Yak-Build-Time。
开发版本（version 含 dev）始终放行；缺失或未知版本按未知版本拦截。
命中拦截会返回 429 memfit_client_version_limited，文案统一提示"旧版本/未知版本使用量已达上限，请升级"。
填写格式建议：2025-06-01T00:00:00Z 或 2025-06-01T08:00:00+08:00。 留空则只按「缺失版本头/未知版本」拦截，不按时间过滤。

模型用途降级规则

客户端在调用时通过请求头 X-Yak-AI-Model-Usage-Type 上报本次是「高质 intelligent / 快速 lightweight / 视觉 vision」哪一档发起的。
命中规则（用途类型匹配且请求模型 == 源模型）时，服务端会自动把请求模型降级为目标模型，以保护用量。
用途类型选「任意」表示不限 tier。模型名称请填写对外名称（如 memfit-standard-free）。
不添加任何规则即表示关闭降级（保存为空列表）。
